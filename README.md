# Ultimate tic-tac-toe

*…by which I of course mean noughts and crosses :p*

node.js implementation of the game described on
[Ben Orlin’s blog](http://mathwithbaddrawings.com/2013/06/16/ultimate-tic-tac-toe/).

## Installation

```
npm install
node app.js
```
